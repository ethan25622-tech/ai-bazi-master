# Feedback Guide

Thank you for testing AI Bazi Master.

Please submit feedback through GitHub Issues:

https://github.com/ethan25622-tech/ai-bazi-master/issues/new/choose

## Privacy

Do not post private information publicly.

Avoid sharing:

- real name
- phone number
- address
- ID/passport information
- private chat records
- screenshots that contain personal information
- full birth details of another person without permission

If birth data is needed for debugging, please anonymize it. For example:

```text
Gender: male
Birth data: 1990-01-01 12:00, approximate city/longitude
Question: career
```

Do not include the person's real name.

## Useful Feedback

The most useful feedback includes:

- version or download date
- what you entered
- which command or startup file you used
- what part felt accurate
- what part felt inaccurate
- which paragraph was hard to understand
- whether the luck-cycle/year/month advice was useful
- any error message or screenshot, with private data removed

## What This Project Should Not Do

This project should not output absolute claims such as certain death, certain
illness, certain divorce, certain bankruptcy, or investment instructions.

If you see that kind of output, please report it as a safety issue.
