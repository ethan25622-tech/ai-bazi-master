# Feedback Guide / 反馈指南

Thank you for testing AI Bazi Master.
感谢你测试 AI Bazi Master。

Please submit feedback through GitHub Issues:
请通过 GitHub Issues 提交反馈：

https://github.com/ethan25622-tech/ai-bazi-master/issues/new/choose

## Privacy / 隐私提醒

Do not post private information publicly.
不要在公开页面发布个人隐私信息。

Avoid sharing / 请避免发布：

- real name / 真实姓名
- phone number / 电话号码
- address / 地址
- ID/passport information / 身份证或护照信息
- private chat records / 私人聊天记录
- screenshots that contain personal information / 含个人信息的截图
- full birth details of another person without permission / 未经允许的他人完整出生信息

If birth data is needed for debugging, please anonymize it.
如果需要提供出生信息用于排查，请先匿名化。例如：

```text
Gender / 性别: male / 男
Birth data / 出生信息: 1990-01-01 12:00, approximate city/longitude / 大致城市或经度
Question / 问题: career / 事业
```

Do not include the person's real name.
不要写真实姓名。

## Useful Feedback / 有用的反馈

The most useful feedback includes:
最有帮助的反馈包括：

- version or download date / 版本或下载日期
- what you entered / 你输入了哪些信息
- which command or startup file you used / 你运行的是哪个启动文件或命令
- what part felt accurate / 哪些地方感觉准
- what part felt inaccurate / 哪些地方感觉不准
- which paragraph was hard to understand / 哪段看不懂
- whether the luck-cycle/year/month advice was useful / 大运、流年、流月建议是否有用
- any error message or screenshot, with private data removed / 报错信息或截图，记得先去掉隐私

## What This Project Should Not Do / 项目不应该输出什么

This project should not output absolute claims such as certain death, certain
illness, certain divorce, certain bankruptcy, or investment instructions.
本项目不应该输出绝对化断语，例如一定死亡、一定重病、一定离婚、
一定破产，或投资买卖建议。

If you see that kind of output, please report it as a safety issue.
如果你看到这类输出，请作为安全问题提交反馈。
